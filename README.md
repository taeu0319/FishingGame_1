# 🎣 Java 1인칭 선상 낚시 게임 (Java Fishing Tycoon)

Java의 기본 그래픽 라이브러리(`Swing`, `AWT`)를 활용하여 구현한 타이쿤 스타일의 1인칭 낚시 게임입니다. 
단순한 텍스트 기반 낚시를 넘어, **실시간 타이머 이벤트, 타이밍 액션, 기하급수적 인플레이션 성장 시스템**을 도입하여 높은 몰입감을 제공합니다.

## 🚀 프로젝트 개요
* **개발 언어:** Java (JDK 8+)
* **실행 환경:** 데스크톱 (Windows, Mac, Linux 호환)

---

## ✨ 핵심 기능 (Key Features)

### 1. ⚡ 타이밍 액션 시스템 (반사신경 미니게임)
* 물고기의 등급(일반~초월)에 따라 입질 후 챔질해야 하는 **제한 시간(1~3초)이 동적으로 변화**합니다.
* `javax.swing.Timer`를 활용하여 메인 화면의 멈춤 없이 비동기적으로 타이머가 작동하도록 설계했습니다.

### 2. 🦈 보스전 시스템 (거대 상어)
* 0.1%의 확률로 등장하는 거대 상어 출현 시, 게임이 **'보스전 모드'**로 전환됩니다.
* 5초의 제한 시간 동안 10번을 연타해야만 포획할 수 있는 긴장감 넘치는 이벤트를 구현했습니다.
* 포획 성공 시 화면 전체에 화려한 파티클 깜빡임 이펙트가 발생합니다.

### 3. 📈 성장 및 시각적 진화 시스템
* **선박 업그레이드:** 50만 G부터 시작해 비용이 2배씩 증가하며, 레벨에 따라 낡은 뗏목 ➔ 튼튼한 목선 ➔ 초호화 요트로 배의 디자인이 진화합니다.
* **장비 강화:** 레벨이 오를수록 낚싯대의 색상이 변하며, 고급 어종(레어, 전설 등)이 등장할 확률이 점진적으로 증가합니다.

### 4. 🛠️ 자원 관리 (내구도 및 상점)
* 낚시를 할 때마다 장비의 내구도가 감소하며, 내구도가 0이 되면 페널티가 발생합니다.
* 낚은 물고기를 팔아 자산을 모으고, 장비를 수리하거나 강화하는 **순환형 경제 시스템**을 구축했습니다.

## 🎮 실행 방법 (How to Run)

방문자의 환경에 맞게 아래의 세 가지 방법 중 하나를 선택하여 게임을 즐겨보세요!

### 💡 방법 1: 가장 쉬운 방법 (일반 사용자용)
개발 프로그램이 없어도 윈도우/맥 환경에서 바로 실행할 수 있는 방법입니다.
1. 우측의 **[Releases]** 메뉴에서 `FishingGame.jar` 파일을 다운로드합니다.
2. 다운로드한 `.jar` 파일을 **더블 클릭**하면 게임이 바로 실행됩니다!
> ※ 주의: PC에 Java(JRE)가 설치되어 있어야 합니다.

### 💻 방법 2: IDE에서 실행하기 (개발자용 추천)
IntelliJ, Eclipse, VS Code 등의 편집기를 사용 중이시라면 아래 방법을 추천합니다.
1. 초록색 **[<> Code]** 버튼을 눌러 프로젝트를 `ZIP`으로 다운로드하거나 클론(`git clone`)합니다.
2. 사용 중인 IDE에서 프로젝트 폴더를 엽니다.
3. `FishingGame.java` 파일을 열고, 코드 내부의 `main` 메서드 옆에 있는 **초록색 재생 버튼(Run)**을 클릭합니다.

### ⌨️ 방법 3: 터미널 환경 (CLI)
명령어 사용이 익숙하신 분들은 터미널에서 직접 컴파일하여 실행할 수 있습니다.
```bash
# 1. 파일 컴파일
javac FishingGame.java

# 2. 게임 실행
java FishingGame
